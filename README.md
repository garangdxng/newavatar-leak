# Avatar Leak Static Site

Files included:
- `index.html`
- `styles.css`
- `success.html`
- `vercel.json`
- `hero-frame.jpeg`

Deploy in the browser:
1. Create a GitHub repo.
2. Upload these files directly into the repo.
3. Import that repo into Vercel.
4. In Stripe Payment Link settings, set the post-payment redirect URL to `/success` on your live domain.

Current payment link is already embedded in the page.
